# Agenda de Contatos

Aplicação web simples para cadastro de contatos.
O usuário pode adicionar um nome e telefone através de um formulário, e os dados são exibidos dinamicamente em uma tabela na página.

## Funcionalidades

- Adicionar novos contatos
- Exibir contatos em uma tabela
- Interface centralizada e organizada
- Botão com efeito hover
- A tabela só aparece após o primeiro contato ser cadastrado

## Tecnoligias utilizadas

- HTML5
- CSS3
- JavaScript (DOM manipulation)
